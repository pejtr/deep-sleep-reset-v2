## Google Conversion Tracking — Instalace

Bez conversion trackingu Google Ads nemůže optimalizovat. Nutné nastavit.

### Kroky:
1. Google Ads → Tools → Conversions → New Conversion → Website
2. Vytvoř konverzi "Purchase" — hodnota: 25 Kč (nebo dynamická)
3. Zkopíruj Global Site Tag (gtag.js) a Conversion Event snippet
4. Přidej do `client/index.html` (Global Site Tag) a na `/thank-you` stránku (Conversion Event)

**Global Site Tag** (do `<head>` v index.html):
```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-XXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-XXXXXXXXX');
</script>
```

**Conversion Event** (na /thank-you stránce):
```html
<script>
  gtag('event', 'conversion', {
    'send_to': 'AW-XXXXXXXXX/XXXXXXXXX',
    'value': 25.0,
    'currency': 'CZK',
    'transaction_id': ''
  });
</script>
```

> Pošli mi Google Ads ID (AW-XXXXXXXXX) a já přidám tracking do kódu.
